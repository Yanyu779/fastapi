# GitHub推送命令

## 如果你有Personal Access Token:
cd /workspace/fullstack-app
git remote set-url origin https://YOUR_TOKEN@github.com/Yanyu779/flask-react-fullstack.git
git push -u origin main

## 或者创建新的Personal Access Token:
1. 访问: https://github.com/settings/tokens
2. 点击 "Generate new token (classic)"
3. 选择权限: repo (Full control of private repositories)
4. 复制生成的token
5. 替换上面命令中的 YOUR_TOKEN
